# Bee Punk
A loja que reconhece a moda como um ato de resistência e cultural. 

# Sobre
Bee Punk é uma loja fictícia de roupas da subcultura punk criada com o objetivo de reconhecer a subcultura e o movimento punk em si, compreendendo a influência dele não apenas na moda mas também nos moldes da contemporaneidade. 

E, além disso, essa loja fictícia faz parte de um projeto de desenvolvimento web que estou realizando com o objetivo de evoluir constantemente nesse campo da TI — abrangendo a estruturação e estilização com as linguagens HTML5 e CSS3, respectivamente; agilização do desenvolvimento com o framework Bootstrap, que também contribuirá para uma melhor responsividade do site habilidades em design UI/UX, prezando por uma boa experiência do usuário ao ele acessar o site; e criatividade, usando recursos de design e ideias a fim de criar própria identidade visual da plataforma.
